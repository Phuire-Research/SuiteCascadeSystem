# S1 — Greeting Skill

**Skill ID**: S1
**Script**: `script.ts`
**Execution**: `npx tsx Skills/S1-Greeting/script.ts [--name <name>] [--format <text|json>]`

---

## Informative

The Informative aspect specifies exactly how to invoke the CLI:

```
npx tsx Skills/S1-Greeting/script.ts --name "World" --format json
```

**Parameters**:
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--name` | string | `"World"` | The name to greet |
| `--format` | `text` or `json` | `text` | Output format |

**Return**: The script writes a greeting file to `output/greeting.txt` (text) or `output/greeting.json` (json) and prints the file path to stdout.

---

## Actionable

**Text format return**:
```
Hello, {name}! Generated by Suite 8 at {ISO timestamp}.
```

**JSON format return**:
```json
{
  "greeting": "Hello, {name}!",
  "suite": "Fresh Slate",
  "skill": "S1-Greeting",
  "timestamp": "{ISO timestamp}",
  "format": "json"
}
```

---

## Concluder

| Concluder | Shell | Verifies |
|-----------|-------|----------|
| Exists | `test -f output/greeting.txt && echo ok` | File was produced |
| Char | `wc -c output/greeting.txt` | Non-zero bytes written |

The Informative tells the agent what to run. The Actionable tells the agent what to expect back. The Concluder verifies the Lambda-event occurred.
